# Issue Tracker: GitHub

Issues and PRDs live in GitHub Issues for `Hackerismydream/CodeCairn`. Use the
`gh` CLI and infer the repository from `origin`.

External pull requests are not a triage request surface. Collaborator pull
requests follow the normal review workflow.

When a skill says to publish, read, label, comment on, or close a ticket, use the
corresponding `gh issue` command. GitHub issues and pull requests share a number
space, so resolve ambiguous references before mutating them.
