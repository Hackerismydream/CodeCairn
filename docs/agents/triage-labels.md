# Triage Labels

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation required |
| `needs-info` | `needs-info` | Waiting for reporter information |
| `ready-for-agent` | `ready-for-agent` | Fully specified and independently executable |
| `ready-for-human` | `ready-for-human` | Human judgment or action required |
| `wontfix` | `wontfix` | Intentionally not actioned |
