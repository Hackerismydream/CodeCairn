# Scope Is a Coding-Agent Memory Runtime

Version 1 includes passive Codex and Claude Code import, Agent Trace
normalization, task-shaped segmentation, five Coding Memory types, Markdown
truth, SQLite state, LanceDB indexing, Recall Context, CLI and HTTP access, and
reproducible evaluation.

Agent execution, live hooks, hidden prompt injection, dashboards, cloud
multi-tenancy, and broad knowledge management are out of scope.
