# Version One Ships in Three Milestones

M1 delivers Codex import, the Agent Trace contract, stable Task Episodes,
Markdown truth, SQLite Import Ledger, and shared CLI/HTTP import and list paths.

M2 delivers Hybrid Compression, the Evidence Gate, five Coding Memory types,
Claude Code import, LanceDB, Mini Cascade, hybrid recall, and Recall Context.

M3 delivers LoCoMo, retrieval evaluation, the 20-task memory-on/off experiment,
recovery tests, health diagnostics, and a reproducible metrics report.
